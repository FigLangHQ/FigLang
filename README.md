# FigLang
A natural programming language that reads like English and thinks like a human.

## Installation
1. Install Python 3 from [python.org](https://python.org)
2. Download all files from this repository and put them in a folder
3. Install the [FigLang VSCode Extension](https://marketplace.visualstudio.com/items?itemName=FigLangHQ.figlang)
4. Open any `.fig` file in VSCode and press the Run button â€” that's it!

## Example
```
name is "World"
say "Hello, " and name
```
